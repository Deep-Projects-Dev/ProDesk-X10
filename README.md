# ProDesk

ProDesk is a comprehensive productivity app designed specifically for CBSE Class 10 students. It enhances study efficiency and overall well-being by integrating multiple proven scientific study techniques, minimizing the friction to start studying, and promoting healthy planning habits.

## Features

- **Calendar Integration**: Plan your study schedule and track important dates with an intuitive calendar interface.
- **NCERT Study Materials**: Access and organize CBSE Class 10 NCERT content for easy reference and study.
- **Revision Tools**: Utilize spaced repetition and active recall techniques to reinforce learning and improve retention.
- **Todo Lists**: Create and manage daily tasks to stay organized and focused on your goals.
- **Healthy Planning**: Incorporate breaks, exercise reminders, and balanced study sessions to maintain physical and mental health.

## Scientific Study Techniques

ProDesk incorporates evidence-based methods to boost productivity:
- **Spaced Repetition**: Review material at optimal intervals to enhance long-term memory.
- **Active Recall**: Test yourself regularly to strengthen understanding.
- **Pomodoro Technique**: Work in focused bursts with scheduled breaks to maintain concentration.
- **Goal Setting**: Break down large tasks into manageable steps with clear deadlines.

## Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/your-username/prodesk.git
   ```
2. Navigate to the project directory:
   ```
   cd prodesk
   ```
3. Install dependencies:
   ```
   npm install
   ```

### Running the App
1. Start the development server:
   ```
   npm run dev
   ```
2. Open your browser and navigate to `http://localhost:5173`
